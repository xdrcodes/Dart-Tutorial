# LOOP CONTROL STATEMENTS
Loops are of two types:
- Definite
  - `for` Loop
- Indefinite
  - `while` Loop
  - `do - while` Loop
## ITERATORS SYNTAX
### `for` Loop
```
for(var i = 0; i < 4; i++) {
    print("Hello");
  }
```
### `while` Loop
```
var i = 0;
  while(i < 4) {
    print("Hello");
    i++;
  }
```
### `do - while` Loop
```
var i = 0;
  do {
    print("Hello");
    i++;
  } while(i < 4);
```
## LOOP STRUCTURE
- Counter Variable
- Condition Check
- Increment/Decrement the counter
